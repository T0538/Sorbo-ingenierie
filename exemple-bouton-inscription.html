Code HTML à copier-coller :

<button class="inscription-btn" onclick="openInscriptionForm('Nom Formation', 'Durée', 'Prix', 'ID')">
    <i class="fas fa-graduation-cap"></i> S'inscrire à cette formation
</button>

Code JavaScript à ajouter :

<script>
function openInscriptionForm(formationName, duration, price, id) {
    const params = new URLSearchParams({
        name: encodeURIComponent(formationName),
        duration: encodeURIComponent(duration),
        price: encodeURIComponent(price),
        id: encodeURIComponent(id)
    });
    
    window.open(`inscription-formation.html?${params.toString()}`, '_blank');
}
</script>
